Option 1:
Get Minecraft Education Edition - you can log in using your [child's] school login.
Download the school world file


Option 2:
To connect to the school Minecraft server on Java Edition:
- Add server MAcademy.minehut.gg
- If no one is using the server, you will find yourself in a public lobby (Mearns Academy is not responsible for public lobby servers). - If you are in a lobby, type: /join MAcademy - If you are still in the lobby after a couple of minutes, type /join MAcademy again.

For Bedrock users (PC): - Add server bedrock.minehut.com on port 19132.
- You will find yourself in a public lobby (Mearns Academy is not responsible for public lobby servers). - Type: /join MAcademy - If you are still in the lobby after a couple of minutes, type /join MAcademy again.
